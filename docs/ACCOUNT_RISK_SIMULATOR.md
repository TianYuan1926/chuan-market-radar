# Account Risk Simulator

## 定位

`src/lib/risk/account-risk-simulator.ts` 是账户级风险模拟器，只读、展示用、辅助判断用。

它必须在后端已经给出结构化计划之后使用，不能生成方向、入场、止损、目标，也不能把 WAIT / WATCH 升级成 `TRADE_PLAN_READY`。

## 默认用户风险约束

- 总资金：`1500 USDT`
- 单币初始保证金：总资金 `3%`，即 `45 USDT`
- 仓位语境：`cross`
- BTC / ETH：固定 `150x`
- 其他山寨币：必须传入交易所最高杠杆；缺少时输出 `unknown` / `not_available`
- 最低结构盈亏比：`3:1`
- 最大止损亏损规则：默认不超过总资金 `3%`

## 输出内容

模拟器输出：

- 计划名义仓位估算
- 初始保证金估算
- 止损亏损估算
- 目标收益估算
- 杠杆风险等级
- 全仓高杠杆距离风险估算
- 结构盈亏比是否达标
- 止损是否有结构意义
- 最大亏损是否超过用户规则

## 硬边界

- 不接交易所 API。
- 不读取 secret。
- 不下单。
- 不生成交易计划。
- 不修改 scan / analysis / strategy / backtest 排序或门禁。
- 不把山寨币未知杠杆猜成默认值。
- 不把账户风险模拟结果当成放行依据。

## 爆仓距离说明

全仓强平不是 isolated 精确公式，真实强平依赖账户余额、维持保证金、未实现盈亏、交易所规则和其它仓位。

因此本模块只输出 `cross_margin_estimate_only`：用“价格反向波动吃掉初始保证金的距离”提示止损是否过度接近高杠杆危险区，不能当作交易所精确强平价。

## 使用顺序

```text
后端结构计划
-> 结构止损和目标
-> 结构盈亏比 >= 3:1
-> 风控门禁
-> account risk simulator 只读换算
```
